## #chat

A web interface for IRC.

![image of #chat](http://imgur.com/zVqHKYX.png)
![image of #chat](http://imgur.com/ihqUNCd.png)

### Tech

- Express
- Socket.io/WebSockets
- React

### Requirements

- Node 8+
- `yarn`

### Getting started

0. Clone the repository
```bash
git clone https://github.com/tsoporan/chat.git chat
```

1. Deps
```bash
cd chat && yarninstall
```

2. Start 'er up:

```bash
yarn start
```
